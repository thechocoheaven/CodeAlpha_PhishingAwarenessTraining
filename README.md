# Phishing Awareness Training

## Description
This project was developed as part of the CodeAlpha Cyber Security Internship.

The presentation aims to educate users about phishing attacks, social engineering tactics, fake websites, suspicious emails, and best practices to stay secure online.

## Topics Covered

- Introduction to Phishing
- Types of Phishing Attacks
- Common Signs of Phishing Emails
- Fake Websites and URL Analysis
- Social Engineering Techniques
- Real-World Phishing Examples
- Best Practices for Protection
- Interactive Quiz and Answers
